# ✅ Class 8: Arrays in C (Beginner-Friendly)

This repository contains beginner-friendly content and code examples to help students understand **arrays in C programming**.

## 🎯 Objectives of Class 8

By the end of this class, students will be able to:
- Understand what an **array** is and why it’s useful
- Learn how to declare and use arrays
- Access elements using **index**
- Work with arrays using **loops**
- Write C programs using arrays

---

## 🧠 What is an Array?

An **array** is a **collection of variables of the same type**, stored together under one name.

---

## 📂 Contents

### 🔹 One-Dimensional Array (`1D_Array.c`)
- Stores values in a single row
- Accessed using one index

### 🔸 Two-Dimensional Array (`2D_Array.c`)
- Think of it like a table (rows × columns)
- Accessed using two indexes: `row` and `column`
